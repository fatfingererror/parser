### Running the app
From the directory where the code is cloned run:
`python parser`

e.g code is in /Users/dev/parser/
* cd /Users/dev/
* python parser

### Running the tests
From the directory where the code is cloned run:
`python -m unittest parser.tests.test_parser`

e.g code is in /Users/dev/parser/
* cd /Users/dev/
* python -m unittest parser.tests.test_parser
